📚 知识库配置详细指南

第一步：登录 Coze 平台

访问 https://console.coze.cn 登录您的账号

第二步：进入知识库管理

在左侧导航栏找到「知识库」或「Knowledge Base」选项，点击进入

第三步：创建知识库

  1. 点击「创建知识库」按钮
  2. 输入知识库名称：校园校小助知识库
  3. 选择知识库类型：文档型知识库
  4. 点击「创建」

第四步：上传文档

  1. 进入知识库详情页
  2. 点击「上传文档」
  3. 选择您的校园文档（支持 PDF、Word、TXT、Markdown 等格式）
  4. 等待文档解析完成

第五步：绑定到工作流

  1. 返回您的工作流项目
  2. 找到「工作流设置」或「Workflow Settings」
  3. 在「知识库绑定」中选择您创建的知识库
  4. 点击「保存」
💡 提示

上传的文档越多，校小助的回答越准确！建议上传：

  • 校园通知和公告
  • 校规校纪
  • 办事指南和流程
  • 图书馆开放时间
  • 食堂作息
  • 考试安排
  • 奖学金评定标准
