### 第二步：在 Coze 平台配置知识库

**请按以下步骤操作：

#### 1️⃣ 找「知识库」入口

在 Coze 平台左侧导航栏，找：
- **「知识库」或「Knowledge Base」

#### 2️⃣ 创建知识库并上传文档

1. 点击「创建知识库」
2. 命名：`校园校小助知识库`
3. 上传您的校园文档
4. 等待解析完成

#### 3️⃣ 获取知识库 ID

1. 进入您刚创建的知识库
2. 在 URL 或设置中找到 ID（格式类似 `kb_xxxxxx`）
3. 复制这个 ID

#### 4️⃣ 配置到项目中（使用我们的 UI 配置更简单

打开 `ui/config.html`，找到「知识库」Tab，粘贴 ID
